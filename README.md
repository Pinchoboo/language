# MPL prototype
Compiler and some example programs in MPL. 

The rust source code is spaghetti without comments.

# Requirements
llvm12

clang and llvm-opt-12 in path

# Usage
cargo run -- path/to/file.mpl

to run the benchmarks run: cargo test
